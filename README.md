# React ToDo App

A simple and clean ToDo application built with **React**.  
You can add tasks, mark them as completed, edit them, delete them, and all changes are automatically saved in **localStorage**.

This project is focused on practicing core React concepts such as:

- `useState`
- `useEffect`
- Props and component communication
- Rendering lists
- Conditional rendering
- LocalStorage persistence
- Component structure
